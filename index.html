DFSPH Fluid Simulation

  • Implemented as part of my Master's Thesis
  • C++ solver written for my PBA application
  • Solves 2 PPE's, explicit viscosity, CFL condition, Occupancy Grid
  • Marching Tetrahedra to generate volume and GI for rendering
  • Python scripting for the simulation set up and job/assets mangement using SLURM

*Flyaway particles are from not using velocity clamps. Artifacts are failed render jobs (not re-rendered yet)
